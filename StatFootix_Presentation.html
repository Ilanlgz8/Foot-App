Netlify Functions

football.js

→ football-data.org

Source principale : matchs du jour, classements, buteurs. Retransmet le 429 pour que TanStack garde le cache.

news.js

→ 4 flux RSS

Agrège L'Équipe, RMC, Eurosport, Goal. Parse le XML, extrait titre, image et date.

espn.js

→ ESPN API

Scores en temps réel. Whitelist de slugs (fra.1, eng.1…) pour sécuriser le proxy.

apifootball.js

→ api-football.com

Minutes précises des matchs live. Retransmet x-quota-remaining pour le basculement heuristique.

Hooks React

localCache.js
Cache localStorage avec TTL. Fallback données périmées si API down.
useTodayMatches
Matchs du jour + navigation par date. TTL 2min si live, 1h sinon.
useLiveMatches
IN_PLAY + PAUSED en parallèle. Refetch 30s si actifs, 5min sinon.
useLiveMinute
Minute précise via api-football. 4 fenêtres de polling ciblées.
useEspnScores
Scores ESPN par fuzzy matching des noms d'équipe (5 premiers chars).
useNews
RSS filtré (mots-clés foot) + déduplication par similarité de titres.
useStandings
Classement simple OU multi-groupes (CdM/LDC) selon la réponse API.
useMatchDetails
Buts, cartons, arbitre, stade. Chargé uniquement quand la modal est ouverte.
useTeamForm
Forme 5 derniers matchs (V/N/D) pour toutes les équipes.
useScorers
Top 20 buteurs d'une compétition. Chargé par le composant Classement.
useWcKnockout
Matchs phase finale CdM, triés dans l'ordre du bracket.
useRecentResults / useMatchs
Résultats J-4 et matchs par compétition groupés par journée.
